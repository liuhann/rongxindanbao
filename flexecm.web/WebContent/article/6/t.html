六边形构建

首先，我们要构建一个六边形。 

我们的立方体期待是由3个菱形拼装而成， 因此我们创建一个由3个div组成的层

<div class="honeycomb yellow">
    <div class="bktop"></div>
    <div class="bkleft"></div>
    <div class="bkright"></div>
 </div>
  
其中bktop的颜色和bkleft、bkright颜色不尽相同，为了体现阴影效果 

.honeycomb {
  position: absolute;
  left: 100px;
  top: 50px;
}
.honeycomb.yellow .bktop {
  background-color: #FFE318;
}
.honeycomb.yellow .bkleft {
  background-color: #FFCE18;
}
.honeycomb.yellow .bkright {
background-color: #FFCE18;
}

首先是上面的方块， 我们要利用css transform的特性 。其实css transform的很多内容数学性还是比较强的，需要一定的理解。我推荐的最好的讲解地址在 http://css3files.com/transform/

我们要结合 skew和rotate两项处理来完成一个菱形。 

skew就是沿x和y2个方向的拉伸
 skew(30deg, 10deg)

而rotate比较好理解 就是旋转
rotate(45deg)

.bktop {
  width: 88px;
  height: 79px;
  -ms-transform: rotate(-30deg) skew(30deg);
  -moz-transform: rotate(-30deg) skew(30deg);
  -webkit-transform: rotate(-30deg) skew(30deg);
}

因为我们只拉伸了x，因此高度宽度就不一样了，因此div的高度和宽度也不相同
而bkleft和bkbottom还要做平移的操作即
.honeycomb .bkleft {
 width: 88px;
 height: 79px;
 -ms-transform: rotate(90deg) skew(-30deg) translate(10px,39px);
 -moz-transform: rotate(90deg) skew(-30deg) translate(10px,39px);
 -webkit-transform: rotate(90deg) skew(-30deg) translate(10px,39px);
}
.honeycomb .bkright{
  width: 88px;
  height: 79px;
  -ms-transform: rotate(90deg) skew(30deg) translate(-67px,-41px);
  -moz-transform: rotate(90deg) skew(30deg) translate(-67px,-41px);
  -webkit-transform: rotate(90deg) skew(30deg) translate(-67px,-41px);
}

用Jquery结合数据结构动态生成这些方块 

for(var i=0; i<data.length; i++) {
   var n = $('<div class="honeycomb"><div class="bktop"></div><div class="bkleft"></div><div class="bkright"></div><div class="content">听</div></div>')
   n.find(".content").html(data[i].content);
   n.addClass(data[i].type);
   n.css("left", data[i].left);
   n.css("top", data[i].top);
   
   $("body").append(n);
  }

落下效果

首先，我们让方块都在屏幕上方，并且其中的3个部分都有一定的旋转 

function fallDown() {
  $(".honeycomb .bkleft").each(function() {
   var r1 = -Math.random() * 60;
   var r2 = -Math.random() * 15;
   $(this).css("-webkit-transform",  "rotate(" +　(80+r1) +　"deg) skew(-" + (30 + r2) +"deg) translate(-55px,45px)");
  }); 
 
  $(".honeycomb .bkright").each(function() {
   var r1 = Math.random() * 60;
   var r2 = -Math.random() * 15;
   $(this).css("-webkit-transform",  "rotate(" +　(80+r1) +　"deg) skew(-" + (30 + r2) +"deg) translate(-160px,-46px)");
  }); 
  $(".honeycomb").each(function() {
   $(this).data("ot", parseInt($(this).css("top")));
   $(this).css("transform", "translateY(-" + (parseInt($(this).css("top")) + 200 - Math.random()*100) + "px) translateZ(0)");
  });
  setTimeout(okay, 200);
 }

这里我们使所有菱形都有一个随机的旋转，然后在200毫秒之后， 让他们过渡到上一节所设置的css

function okay() { 
  $(".honeycomb, .honeycomb .bkleft, .honeycomb .bkright").addClass("transted");
  $(".honeycomb .bkleft").css("-webkit-transform", "rotate(90deg) skew(-30deg) translate(10px,39px)");
  $(".honeycomb .bkright").css("-webkit-transform", "rotate(90deg) skew(30deg) translate(-67px,-41px)");
  $(".honeycomb").css("transform", "translateY(0px) translateZ(0)");
 }
过渡效果就是我们新增的一个css样式， 它是： 
.transted {
 -webkit-transition: all 2s cubic-bezier(.3,.12,.43,1.24);
 -moz-transition: all 2s cubic-bezier(.3,.12,.43,1.24);
 -ms-transition: all 2s cubic-bezier(.3,.12,.43,1.24);
}

蜂巢切换
最后，我们希望点击一个蜂巢时， 会切换到另一个蜂巢页面 。  

我们编写一个新的缩放的切换效果 
 function zoomOut() {    
  $(".honeycomb").css("transform", "scale(0)");
  setTimeout(function() {
    var i = 1;
   $(".honeycomb").each(function() {
    $(this).css("-moz-transition" , "all .5s ease-in-out " + i*30 + "ms");
    $(this).css("-ms-transition" , "all .5s ease-in-out " + i*30 + "ms");
    $(this).css("-webkit-transition" , "all .5s ease-in-out " + i*30 + "ms");
    $(this).css("transition" , "all .5s ease-in-out " + i*30 + "ms");
    i++
   });
   $(".honeycomb").css("-moz-transform", "scale(1)");
   $(".honeycomb").css("-ms-transform", "scale(1)");
   $(".honeycomb").css("-webkit-transform", "scale(1)");
   $(".honeycomb").css("transform", "scale(1)");
  }, 100);
 }
依次让方块放大显示的效果 

然后给每个方块增加一个点击事件
$(".honeycomb").click(function() {
      $(".honeycomb").remove();
      init(sea2);
      zoomOut();
    });

小结
本文介绍了一种蜂巢展示页面的实现 。核心围绕着 transform属性展开 。 
因为位置变换全部使用 translate来实现的，在手机上也能获得相当平滑的效果。
